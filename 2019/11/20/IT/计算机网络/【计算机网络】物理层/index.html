物理层简介

物理层之间的传输单位是比特（bit）

物理层考虑的是，怎样才能在连接各种计算机的传输媒体上传输数据比特流，描述了传输媒体接口的特性：

  1. 机器特性：接口的尺寸，形状，引脚数等。
  2. 电气特性：电缆上各条线的电压范围
  3. 功能特性：某条线上的电压意义
  4. 过程特性：对于不同功能的各种可能事件的出现顺序

数据通信基础知识

一，信道的分类

  1. 单向通信（单工）：只能向一个方向发送信号，例如：广播，集线器
  2. 双向交替通信（半双工）：可以双向发送，但是不能同时发送
  3. 双向同时通信（全双工）：可以双向并且同时发送，如交换机

二，常用的数字信号编码方式

  1. 不归零制：正电平代表1，负电平代表0
  2. 归零制编码：正脉冲代表1，负脉冲代表0
  3. 曼彻斯特编码：位周期中心向上跳变代表0，位周期中心向下跳变表示1
  4. 差分曼彻斯特：在每一位的中心处始终都有跳变。位开始边界有跳变代表0，位开始边界没有跳变代表1

物理层下面的传输媒体

一，引导型传输媒体（沿着媒介传播）

  1. 双绞线
  2. 同轴电缆
  3. 光缆

二，非引导型传输媒体（空间自由传播）

  1. 波

信道复用技术

使用信道复用技术可以节约信道，经济实惠

频分复用

不同频率代表着不同的发送者

时分复用

把帧分成多个段，不同段代表着不同的发送者

码分复用（码分多址）

每一个比特时间在划分为m个短的间隔，称为码片。各个用户使用经过特殊挑选的不同码片序列，并且相互正交（积为0），因此个用户之间不会产生干扰。
