JAVA的一些冷知识

  • java applet 是关于页面显示的程序
  • JAVA原名是“Oak”，橡树的意思，但后来发现有已经有编程语言叫做Oak，所以改名为JAVA

JAVA语言发展情况

版本 年份 新特性
1.0 1996 语言本身
1.1 1997 内部类
1.2 1998 strictfp修饰符
1.3 2000 无
1.4 2002 断言
5.0 2004 泛型类，for each，循环，自动装箱，枚举，静态导入
6 2006 无
7 2011 基于字串的switch，钻石操作符，二进制字面量，异常处理的改进
8 2014 lambda表达式，包含默认方法的接口，流和日期/时间库
